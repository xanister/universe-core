## Syncing Assets

Assets are synced from the Universal LPC Spritesheet repository using:

```bash
npm run sync-lpc-assets
```

This script:

1. Fetches the repository structure via GitHub API
2. Downloads all PNG assets
3. Organizes them by body type (male/female/either)
4. Generates the manifest.json with proper metadata

Set the `GITHUB_TOKEN` environment variable for higher API rate limits.

## Sprite Format

All LPC sprites follow the standard format:

- **Frame Size**: 64x64 pixels
- **Image Size**: 832x1344 pixels (13 columns x 21 rows)
- **Animations**: spellcast, thrust, walk, slash, shoot, hurt
- **Directions**: up, left, down, right (4 rows per animation)
