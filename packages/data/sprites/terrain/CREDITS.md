# Terrain Assets Credits

## Blob-47 Tilesets

All tilesets in the `blob47/` directory are derived from the LPC (Liberated Pixel Cup) terrain-v7 spritesheet, converted to blob-47 format (7x8 grid, 50 tiles) via mini-tile quadrant compositing.

### Source

- **Original Asset**: [LPC Terrains](https://opengameart.org/content/lpc-terrains)
- **License**: CC-BY-SA 3.0 / CC-BY-SA 4.0
- **Original Authors**: Various contributors to the Liberated Pixel Cup

### License Compliance

When using these assets, you must:
1. Credit the original LPC Terrains authors
2. License derivative works under CC-BY-SA 3.0 or CC-BY-SA 4.0
3. Link to the original source when possible

## Legacy Assets (Archived)

The following legacy assets have been replaced and removed:
