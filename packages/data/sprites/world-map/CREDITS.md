## RPG Maker World Map Tileset

- **File:** `rpgmaker-world.png`
- **Dimensions:** 768 × 768 pixels
- **Source:** [RPG Maker Web - How to (mis)Use World Map Tiles](https://www.rpgmakerweb.com/blog/how-to-mis-use-world-map-tiles-for-anything-other-than-world-maps)
- **License:** RPG Maker RTP (check RPG Maker EULA for usage terms)
- **Author:** RPG Maker / Kadokawa

### Description

RPG Maker-style world map tileset containing:
- Castles and fortresses (various styles: stone, snow, golden-roofed)
- Towns and villages
- Mountains (rocky, volcanic, snowy)
- Terrain features (trees, oases, ponds)
- Structures (pyramids, igloos, temples)
- Islands and floating landmasses
- Decorative elements (rainbows, clouds)
